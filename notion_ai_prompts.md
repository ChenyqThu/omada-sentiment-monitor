# Notion AI Column Prompts 指南

## 概述

如果您选择使用 Notion AI Column 而不是我们的 AI 分析器，以下是为每个分析字段推荐的 Prompt。这些 Prompt 已经针对 Reddit 舆情分析进行了优化。

## AI 分析字段和对应 Prompts

### 1. 情感倾向 (AI 情感分析)

**字段类型**: Select (选项: 正面, 负面, 中性)

**Prompt**:
```
分析这个 Reddit 帖子的整体情感倾向。考虑标题、正文内容和主要评论。

请返回以下选项之一:
- 正面: 表达积极、满意、推荐的情感
- 负面: 表达不满、抱怨、批评的情感  
- 中性: 客观陈述、询问问题、中立讨论

重点关注对 TP-Link/Omada 产品的态度。
```

### 2. 情感分数 (情感分数)

**字段类型**: Number

**Prompt**:
```
为这个 Reddit 帖子的情感倾向打分，范围从 -1.0 到 1.0：

- 1.0: 非常正面 (强烈推荐、极度满意)
- 0.5: 正面 (满意、推荐)
- 0.0: 中性 (客观、中立)
- -0.5: 负面 (不满、抱怨)
- -1.0: 非常负面 (强烈批评、极度不满)

只返回数字，保留一位小数。
```

### 3. 置信度 (置信度)

**字段类型**: Number

**Prompt**:
```
评估情感分析的置信度，范围从 0.0 到 1.0：

- 0.9-1.0: 情感表达非常明确
- 0.7-0.9: 情感表达比较明确
- 0.5-0.7: 情感表达一般明确
- 0.3-0.5: 情感表达不太明确
- 0.0-0.3: 情感表达很模糊

考虑因素：用词强烈程度、表达明确性、上下文清晰度。
只返回数字，保留一位小数。
```

### 4. 关键词 (AI 关键词)

**字段类型**: Multi-select

**Prompt**:
```
从这个 Reddit 帖子中提取 5-10 个最重要的关键词。

重点提取：
1. 产品型号 (如: EAP610, EAP615, OC200, Archer AX73)
2. 技术术语 (如: mesh, WiFi 6, PoE, VLAN)
3. 问题类型 (如: 连接问题, 配置难题, 性能问题)
4. 竞品品牌 (如: Ubiquiti, Aruba, Cisco)
5. 应用场景 (如: 小型办公室, 家庭网络, 企业部署)

每个关键词用逗号分隔，不要使用引号。
```

### 5. 主题分类 (AI 主题分类)

**字段类型**: Multi-select

**Prompt**:
```
将这个 Reddit 帖子分类到以下一个或多个主题：

产品相关：
- 产品推荐
- 产品比较
- 新品发布
- 固件更新

技术相关：
- 安装配置
- 故障排除
- 性能优化
- 网络设计

用户体验：
- 用户评价
- 使用心得
- 购买建议
- 售后服务

竞争分析：
- 竞品对比
- 品牌讨论
- 市场趋势

每个分类用逗号分隔。
```

### 6. AI 摘要 (AI 摘要)

**字段类型**: Text

**Prompt**:
```
为这个 Reddit 帖子写一个简洁的摘要（50-100字）：

摘要应包含：
1. 主要问题或讨论点
2. 涉及的产品型号（如有）
3. 关键结论或建议
4. 情感倾向的简要说明

用中文撰写，语言简洁专业。
```

### 7. 产品型号 (产品型号)

**字段类型**: Multi-select

**Prompt**:
```
识别这个 Reddit 帖子中提到的所有 TP-Link 产品型号：

常见型号包括：
- EAP 系列: EAP610, EAP615, EAP650, EAP660, EAP670, EAP680
- Archer 系列: AX73, AX55, AX53, AX23, AX21
- Deco 系列: X60, X55, X50, M5, M9
- OC 控制器: OC200, OC300
- 交换机: TL-SG系列

只提取确实存在的型号，每个型号用逗号分隔。
```

### 8. 问题类型 (问题类型)

**字段类型**: Multi-select

**Prompt**:
```
识别这个 Reddit 帖子中讨论的问题类型：

技术问题：
- 连接问题
- 配置问题
- 性能问题
- 兼容性问题
- 固件问题

使用问题：
- 安装困难
- 操作复杂
- 功能缺失
- 界面问题

其他：
- 购买咨询
- 产品推荐
- 对比评测
- 一般讨论

每个类型用逗号分隔。
```

### 9. 竞品提及 (竞品提及)

**字段类型**: Multi-select

**Prompt**:
```
识别这个 Reddit 帖子中提到的竞争品牌：

主要竞品：
- Ubiquiti (UniFi)
- Aruba
- Cisco
- Netgear
- ASUS
- Linksys
- D-Link
- Ruckus

只提取确实被提及的品牌，每个品牌用逗号分隔。
```

### 10. 优先级 (优先级)

**字段类型**: Select (选项: 高, 中, 低)

**Prompt**:
```
根据以下标准为这个 Reddit 帖子设置优先级：

高优先级：
- 严重负面情感 (情感分数 < -0.5)
- 高影响力用户 (高 karma 或版主)
- 广泛传播 (高点赞/评论数)
- 竞品对比中处于劣势

中优先级：
- 轻微负面或中性情感
- 一般技术问题
- 产品咨询

低优先级：
- 正面评价
- 一般性讨论
- 低互动量

返回：高、中、低 之一。
```

## 使用建议

### 1. 设置步骤
1. 在 Notion Database 中为每个字段添加 AI Column
2. 将对应的 Prompt 粘贴到 AI Column 设置中
3. 测试 Prompt 效果并根据需要调整

### 2. 优化建议
- 根据实际数据质量调整 Prompt
- 定期检查 AI 分析结果的准确性
- 可以结合人工审核提高质量

### 3. 成本考虑
- Notion AI 按使用量计费
- 建议先在少量数据上测试
- 可以选择性启用某些字段以控制成本

### 4. 结合使用
您可以：
- 使用我们的 AI 分析器进行批量处理
- 使用 Notion AI Column 进行人工审核和补充
- 两种方式结合使用以获得最佳效果

## 注意事项

1. **Prompt 长度**: Notion AI Column 有字符限制，如果 Prompt 太长可能需要简化
2. **语言一致性**: 建议在 Prompt 中明确指定返回语言（中文或英文）
3. **格式要求**: 确保返回格式与字段类型匹配
4. **更新维护**: 定期根据新产品和市场变化更新 Prompt

## 测试用例

您可以用以下示例帖子测试 Prompt 效果：

**正面示例**:
"刚升级到 TP-Link EAP610，WiFi 6 的速度真的很快！安装也很简单，推荐给大家。"

**负面示例**:
"EAP615 又断线了，这已经是这个月第三次了。考虑换成 Ubiquiti。"

**中性示例**:
"有人用过 Omada OC200 控制器吗？想了解一下配置复杂度。"
